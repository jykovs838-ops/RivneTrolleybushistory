# rivnetrolleybushistory

Це репозиторій, присвячений історії тролейбусного транспорту міста Рівне.

## Про проект

Проект містить інформацію та документацію про розвиток тролейбусної системи в Рівному, включаючи історичні дані, фотографії та архівні матеріали.

## Вміст

- Історія розвитку тролейбусного транспорту в Рівному
- Архівні документи та фотографії
- Інформація про маршрути та розписи
- Дані про рухомий склад
